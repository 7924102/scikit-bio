bipy.core.sequence.NucleotideSequence.is_reverse_complement¶

NucleotideSequence.is_reverse_complement(other)¶

Return True if other is the reverse complement of self

Returns:

bool

True if other is the reverse complement of self and False otherwise.
Raises:

bipy.core.exception.BiologicalSequenceError

If a character is present in other that is not in the self.complement_map.
