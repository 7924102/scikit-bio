Exceptions (bipy.core.exception)¶

This module defines custom exception classes used throughout the core bipy codebase.

Exceptions¶

BiologicalSequenceError
DistanceMatrixError General error for distance matrix validation failures.
DistanceMatrixFormatError Error for reporting issues in distance matrix file format.
IDMismatchError(actual, expected) Error for reporting a mismatch between IDs.
MissingDataError(actual, expected) Error for reporting missing data lines during parsing.
MissingHeaderError() Error for reporting a missing ID header line during parsing.
MissingIDError(missing_id) Error for ID lookup that doesn’t exist in the distance matrix.
